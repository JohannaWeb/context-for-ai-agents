# Major Project: Project Falcon

Project Falcon is a developer communication and identity platform.

Initial concept:

Discord‑like desktop application built on AT Protocol.

Architecture evolved into a distributed identity and trust
infrastructure.

------------------------------------------------------------------------

# Falcon Architecture

Microservices: - falcon-core - falcon-gateway - falcon-web -
siv-service - trust-service

Backend: - Java 25 - Spring Boot 4 - Project Loom (virtual threads)

Real‑Time Layer: AT Protocol Jetstream ingestion.

Performance target: \~60 ms ingestion latency

Caching: Caffeine caching (\~5 ms hits)

Frontend: - Electron - TypeScript - React - Vite

Infrastructure: - Kubernetes - Observability stack - Zero‑trust gateway
